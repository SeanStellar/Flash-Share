# Flash Share

一个无需账号、无需上传云盘的 Windows 文件直传工具。两台电脑运行同一个 `Flash Share.exe`，即可发送文件或整个文件夹。当前版本为 `1.1.0`，构建好的程序就在本文件夹根目录。

## 1.1.0 新功能

- 可直接从 Windows 资源管理器将文件或文件夹拖入发送列表。
- 可在列表中按住鼠标左键拖过多行进行批量选择，再点击“移除所选”或按 `Delete` 删除。
- 接收端确认传输成功后，已发送的文件和文件夹会自动从发送列表移除；传输失败或取消时仍会保留。
- 手动输入地址时兼容 `IP:端口` 格式，也可以继续在两个输入框中分别填写。

## 使用方法

1. 在接收电脑上双击 `Flash Share.exe`，切换到“接收文件”。
2. 选择保存位置，点击“开始接收”。如果 Windows 防火墙弹出提示，请允许“专用网络”访问。
3. 在发送电脑上运行同一个 EXE，点击“扫描局域网”，或手动填写接收端显示的 IP 和端口。
4. 填写接收端显示的临时连接码，通过按钮或直接拖入文件/文件夹，然后点击“开始发送”。

## 网络范围

- 同一 Wi-Fi、同一路由器或同一局域网：直接使用，速度主要取决于局域网带宽和硬盘速度。
- 不同地点、跨公网：程序支持手动填写 IP，但接收端需要配置路由器端口映射（默认 TCP/UDP `48231`），或者先使用 Tailscale、ZeroTier、蒲公英等组网工具让两台电脑互相可达。
- 本工具不包含公共中转服务器，因此不会把文件上传到第三方云端。

## 安全与可靠性

- 使用随机临时连接码派生 AES-256-GCM 会话密钥，文件内容和控制信息均加密传输。
- 每个文件完成后进行 SHA-256 完整性校验。
- 接收端会阻止路径穿越；目标存在同名文件/文件夹时会自动添加序号，不覆盖原文件。
- 未完成的临时文件会被清理。

## 从源码重新构建

在 PowerShell 中运行：

```powershell
.\build.ps1
```

生成文件位于本文件夹根目录的 `Flash Share.exe`（`dist` 中也会保留一份）。
